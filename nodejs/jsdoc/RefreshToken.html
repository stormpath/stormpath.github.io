Class: RefreshToken

RefreshToken

new RefreshToken(refreshTokenResource)

Encapsulates a Stormpath OAuth Refresh Token Resource. For full documentation of this resource, please see REST API Reference: Refresh Token. For a high-level overview of token management, please see Introduction to Token-Based Authentication.

This class should not be manually constructed. It should be obtained from one of these methods:

  • Client.getRefreshToken()
  • Account.getRefreshTokens()

To revoke a refresh token, invoke the delete() method on an instance of this class.

refreshTokenResource Object

The JSON representation of this resource, retrieved the Stormpath REST API.
