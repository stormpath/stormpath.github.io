Class: ApplicationAccountStoreMapping

ApplicationAccountStoreMapping

new ApplicationAccountStoreMapping(accountStoreMappingResource)

This object encapsulates an Application Account Store Mapping, which represents the link between an Application and an Account Store, such as a Directory, Group, or Organization. For full documentation of this resource, please see REST API Reference: Account Store Mapping.

This class should not be manually constructed. It should be obtained from one of these methods:

  • Application.createAccountStoreMapping()
  • Application.getAccountStoreMappings()

For more information about account store mappings, please see Modeling Your User Base.
