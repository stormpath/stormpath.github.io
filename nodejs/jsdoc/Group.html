Class: Group

Group

new Group(groupResource)

Encapsulates a Group resource. For full documentation of this resource, please see REST API Reference: Group.

This class should not be manually constructed. It should be obtained from one of these methods:

  • Client.getGroup()
  • GroupMembership.getGroup()

getAccountMemberships( [collectionQueryOptions], callback)

Retrieves the list of GroupMemberships for this group.

Parameters:
Name Type Argument Description
collectionQueryOptions CollectionQueryOptions <optional>

Options for querying, paginating, and expanding the collection. This collection does not support attribute or filter searches.
callback function

The function to call when the operation is complete. Will be called with the parameters (err, CollectionResource). The collection will be a list of GroupMembership objects.
Source:
resource/Group.js
Example
group.getAccountMemberships({ expand: 'account' }, function (err, memberships) {
  memberships.each(function (membership, next) {
    console.log(membership.account.fullName + ' is in this group');
    next();
  });
});

getAccounts(collectionQueryOptions, callback)

Get the account's collection, which is a list of all the accounts in this Group.

Parameters:
Name Type Description
collectionQueryOptions CollectionQueryOptions

Options for querying, paginating, and expanding the collection. This collection supports filter searches and the following attribute searches: createdAt, email, givenName, middleName, modifiedAt, surname, status, username.
callback function

The function to call when the operation is complete. Will be called with the parameters (err, CollectionResource). The collection will be a list of Account objects.
Source:
resource/Group.js
Example
var query = {
  email: 'foo@example.com'
};

group.getAccounts(query, function (err, collection) {
  collection.each(function (account, next) {
    console.log('Found account for ' + account.givenName + ' (' + account.email + ')');
    next();
  });
});
