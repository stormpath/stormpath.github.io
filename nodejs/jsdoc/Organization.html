Class: Organization

Organization

new Organization(organizationResource)

Encapsulates a Organization resource. For full documentation of this resource, please see REST API Reference: Organization.

This class should not be manually constructed. It should be obtained from one of these methods:

  • AccountStoreMapping.getAccountStore()
  • Client.getOrganization()
  • Client.getOrganizations()
  • Directory.getOrganizations()

createAccountStoreMapping(accountStoreMapping, callback)

Creates a mapping between this Organization and an Account Store, the Account Store can be a Directory or Group.

Parameters:
Name Type Description
accountStoreMapping Object
Properties
Name Type Argument Default Description
accountStore Directory | Group | Object

The Account Store to set as the default Account Store for this organization. This can be a materialized Directory, Group, or object literal with an href property that identifies the Account Store.
isDefaultAccountStore Boolean <optional>
 false

Set to true if you want this account store to be the default store where new accounts are placed when calling Organization.createAccount(). If you need to change the default account store in the future, use Organization.setDefaultAccountStore()
isDefaultGroupStore Boolean <optional>
 false

Set to true if you want this account store to be the default location of groups when calling Organization.createGroup(). In this situation, the account store must be a Directory. If you need to change the default group store in the future, use Organization.setDefaultGroupStore()
callback function

The function to call when the operation is complete. Will be called with the parameters (err, OrganizationAccountStoreMapping).
Source:
resource/Organization.js
Examples
client.getDirectory(directoryHref, function(err, directory}{
  organization.createAccountStoreMapping({ accountStore: directory }, function (err, organizationAccountStoreMapping) {
    if (!err) {
      console.log('Directory was mapped to the organization.');
    }
  });
})
var mapping = {
  accountStore: {
    href: 'https://api.stormpath.com/v1/directories/xxx'
  }
};

organization.createAccountStoreMapping(mapping, function (err, organizationAccountStoreMapping) {
  if (!err) {
    console.log('Directory was mapped to the organization.');
  }
});

getAccounts(collectionQueryOptions, callback)

Get the account's collection for this Organization, which is a list of all accounts in all account stores that are mapped to this organization.

Parameters:
Name Type Description
collectionQueryOptions CollectionQueryOptions

Options for querying, paginating, and expanding the collection. This collection supports filter searches and the following attribute searches: createdAt, email, givenName, middleName, modifiedAt, surname, status, username.
callback function

The function to call when the operation is complete. Will be called with the parameters (err, CollectionResource). The collection will be a list of Account objects.
Source:
resource/Organization.js
Example
var query = {
  givenName: 'foo'
};

organization.getAccounts(query, function(err, collection) {
  collection.each(function(account, next) {
    console.log('Found account for ' + account.givenName + ' (' + account.email + ')');
    next();
  });
});

setDefaultAccountStore(accountStore, callback)

Sets the default Account Store for this Organization by setting the isDefaultAccountStore property of the OrganizationAccountStoreMapping that represents the link between this Organization and the account store. If this mapping does not already exist it will be automatically created.

Parameters:
Name Type Description
accountStore Directory | Group | Object

The Account Store to set as the default Account Store for this organization. This can be a materialized Directory, Group, or object literal with an href property that identifies the Account Store.
callback function

The function to call when the operation is complete. Will be called with the parameters (err, OrganizationAccountStoreMapping).
Source:
resource/Organization.js
Examples
var directoryHref; // The HREF of an existing directory.

client.getDirectory(directoryHref, function(err, directory){
  organization.setDefaultAcccountStore(directory, function (err) {
    if (!err) {
      console.log('Directory was set as default account store');
    }
  });
})
var directoryHref; // The HREF of an existing directory.

organization.setDefaultAcccountStore({href: directoryHref}, function (err) {
  if (!err) {
    console.log('Directory was set as default account store');
  }
});
