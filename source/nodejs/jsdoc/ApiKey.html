Class: ApiKey

ApiKey

new ApiKey(apiKeyResource)

Encapsulates an Api Key resource of an Account. For full documentation of this resource, please see REST API Reference: Account API Keys.

These keys can be used to authenticate requests to your web framework. For a high-level overview of this feature, please read Using Stormpath for API Authentication. If you are using Express-Stormath, please read the Authentication Section.

This class should not be manually constructed. It should be obtained from one of these methods:

  • Account.getApiKeys()
  • Application.getApiKey()

The JSON representation of this resource, retrieved the Stormpath REST API.
