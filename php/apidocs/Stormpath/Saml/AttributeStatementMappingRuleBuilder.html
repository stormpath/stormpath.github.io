AttributeStatementMappingRuleBuilder
setName(string $name)

Sets the SAML Attribute name, that when encountered, should have its value applied as Account field values. When this name is encountered when processing a SAML Attribute Statement, its associated value will be set as the value for all Stormpath Account field names specified in the accountAttributes.
