SyncApplicationExtensionsSendPasswordResetEmail Method (IApplication, String, IAccountStore)
Synchronously sends a password reset email to an account in the specified Account Store matching the specified email. If the email does not match an account in the specified Account Store, a ResourceException will be thrown. If you are unsure of which of the application's mapped account stores might contain the account, use the more general SendPasswordResetEmail(IApplication, String) method instead. The email will contain a password reset link that the user can click or copy into their browser address bar.
