Stormpath.SDK.DataStore
IDataStore Interface
Stormpath.SDK.DataStore Namespace
The Stormpath.SDK.DataStore namespace provides classes that handle the underlying communication with the Stormpath REST API.
Interfaces
  InterfaceDescription
IDataStore
A IDataStore is the liaison between client SDK components and the raw Stormpath REST API. It is responsible for converting SDK objects (Account, Application, Directory, etc.) into REST HTTP requests, executing those requests, and converting REST HTTP responses back into SDK objects.
