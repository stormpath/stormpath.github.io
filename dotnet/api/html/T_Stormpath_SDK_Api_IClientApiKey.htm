IClientApiKey Interface

Note: This API is now obsolete.

Represents a Stormpath customer's API-specific ID and secret. All Stormpath REST invocations must be authenticated with a Client API Key.

Methods
  NameDescription
GetId
Gets the public unique identifier. This can be publicly visible to anyone - it is not considered secure information.
GetSecret
Gets the raw secret used for API authentication. NEVER EVER print this value anywhere - logs, files, etc. It is TOP SECRET. This should not be publicly visible to anyone other than the person to which the Client API Key is assigned. It is considered secure information.
IsValid
Checks whether the key/secret pair represented by this Client API Key is valid. This is a client-side check for convenience only. It does not make a request to the Stormpath server.
