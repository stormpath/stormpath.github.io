IClientBuilder Interface
A Builder design pattern used to construct IClient instances.

Unless otherwise specified, an in-memory cache will be used to improve performance, with a default TTL (Time to Live) and TTI (Time to Idle) of 1 hour. Caching can be disabled by passing NewDisabledCacheProvider to SetCacheProvider(ICacheProvider).

Note: The default in-memory cache might not be sufficient for an application that runs on multiple servers, due to cache-coherency issues. If your application runs in multiple instances, consider plugging in a distributed cache like Redis.
