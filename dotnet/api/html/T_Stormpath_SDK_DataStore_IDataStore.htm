IDataStore Interface
A IDataStore is the liaison between client SDK components and the raw Stormpath REST API. It is responsible for converting SDK objects (IAccount, IApplication, IDirectory, etc.) into REST HTTP requests, executing those requests, and converting REST HTTP responses back into SDK objects.
