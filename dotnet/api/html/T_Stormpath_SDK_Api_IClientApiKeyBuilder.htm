Examples
Read an API Key at the specified path:
Copy
IClientApiKey apiKey = ClientApiKeys.Builder()
    .SetFileLocation("path\\to\\apiKey.properties")
    .Build();
Then, create a client:
Copy
IClient client = Clients.Builder()
    .SetApiKey(apiKey)
    .Build();
