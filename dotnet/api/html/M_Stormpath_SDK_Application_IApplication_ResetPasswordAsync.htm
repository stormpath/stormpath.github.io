Verifies the password reset token (received in the user's email) and immediately changes the password in the same request, if the token is valid.

Remarks
Once the token has been successfully used, it is immediately invalidated and can't be used again. If you need to change the password again, you will need to execute SendPasswordResetEmailAsync(String, CancellationToken) again in order to obtain a new password reset token.
