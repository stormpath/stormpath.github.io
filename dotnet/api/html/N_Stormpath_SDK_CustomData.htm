Stormpath.SDK.CustomData Namespace
The Stormpath.SDK.CustomData namespace provides classes that are used to interact with Custom Data resources.
Interfaces
  InterfaceDescription
ICustomData
A dictionary resource embedded within another resource (such as an Account, Group, or Directory) that allows you to store arbitrary name/value pairs.
ICustomDataProxy
Provides access to a proxy that can manipulate the custom data of the parent resource.
