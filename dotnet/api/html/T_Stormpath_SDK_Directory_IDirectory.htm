IDirectory Interface
A directory is a top-level container of Accounts and Groups. Accounts and groups are guaranteed to be unique within a directory, but not across multiple directories.

Remarks
You can think of a Directory as an account 'store'. You can map one or more directories (or groups within a directory) to an Application. This forms the application's effective 'user base' of all Account that may use the application.
