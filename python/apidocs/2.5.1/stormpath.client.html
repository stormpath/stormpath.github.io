Client Object¶

class stormpath.client.Client(base_url=None, cache_options=None, expand=None, proxies=None, user_agent=None, backoff_strategy=None, **auth_kwargs)¶

The Client object is what handles all connections to Stormpath, and is the entry point for all Stormpath usage.

Using the Client instance, you can access all Tenant data, such as Applications, Directories, Organizations, Groups, and Accounts.

More info can be found in our documentation: https://docs.stormpath.com/rest/product-guide/latest/reference.html#rest-api-core-concepts

Examples:

client = Client(id='xxx', secret='xxx')
client = Client(id='xxx', secret='xxx', base_url='https://enterprise.stormpath.io/v1')
client = Client(api_key_id='xxx', api_key_secret='xxx')

Initialize the client by setting the stormpath.data_store.DataStore and stormpath.resources.tenant.Tenant.

The parameters the Client accepts are those used by the stormpath.http.HttpExecutor, stormpath.auth.Auth, and stormpath.data_store.DataStore classes.

Parameters:
  • user_agent (str) – (optional) The custom user agent to set.
  • backoff_strategy – A Function that will return the number of milliseconds to wait before retrying the request. The function must take one parameter which is the number of retries already done. If no function is supplied the default backoff strategy is used (see the stormpath.http.HttpExecutor.pause_exponentially() method).
account_links¶

This property allows you to create links between Accounts (AccountLink) in separate Directories

Note

You cannot iterate over AccountLinks directly from this Client object. This is not supported by the Stormpath API.

Examples:

Creating an AccountLink:

account_link = client.account_links.create('left_account': account1, 'right_account': account2)
account_store_mappings¶

This property allows you to access AccountStoreMapping resources.

Note

You cannot iterate over AccountStoreMappings directly from this Client object. This is not supported by the Stormpath API.

Examples:

Creating an AccountStoreMapping:

mapping = client.account_store_mappings.create({
    'application': application,
    'account_store': account_store,
})

Fetching an existing AccountStoreMapping:

mapping = client.account_store_mappings.get(href)
